
#Сборка проекта осуществляется с помощью gulp
> gulp build

# Для разработки вызвать команду 
> gulp watch
# Эта комманда будет отслеживать измененние файлов и производить пересборку файлов

### Разработка
    Исходные html файлы находятся в директории html_src
    В html_components находяться html компоненты, которые подключаются в основные файлы из директории html_src
        пример подключения: @@include('navbar.html')
    Html-файлы компилируются в корень проекта

    Исходные css файлы находяться в директории css/src/
    Gulp минифицирует их в директорию css/dist/
    В файлы из html_src подключаются файлы из сss/dist/

    Исходные файлы js находятся в js/src/
    Gulp минифицирует их в директорию js/dist/
    В файлы из html_src подключаются файлы из js/dist/


При сборке css и js файлов gulp дописывает к имени файла префикс с версией 


### Интернационализация 
    Для интернационализации используется npm библиотека i18next
    Все переводы на 5 языков находятся в файле js/src/i18n/translations.js
    Для добавления перевода в разметке необходимо указать data-атрибут ``data-translation-id`` элементу, содержимое которого должно переводиться
        например: <p class="px-3 pb-4" data-translation-id="featuresP1">
                        Each player is represented by an in the
                        game, and players could with others through the avatar. could also modify the avatar by
                        different assets to it.
                    </p>

    Затем в переменную translations js/src/i18n/translations.js в объект необходимо добавить перевод для каждого языка

### Сборка прод-версии
    Сборка производиться командой, запущенной в корне дев-версии
    > php -f create_prod.php
    
    Данная команда создаст директорию masdProd/ рядом с директорией дев
    Из прод-версии будут удалены все лишние файлы и файлы исходников
